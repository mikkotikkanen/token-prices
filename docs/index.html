Token Prices

Daily updated LLM pricing data for OpenAI, Anthropic, Google, and OpenRouter.

API Endpoints

JSON data files are available at:

OpenAI

api/v1/openai.json

Anthropic

api/v1/anthropic.json

Google

api/v1/google.json

OpenRouter

api/v1/openrouter.json

NPM Package Coming Soon

npm install token-prices
import { getModelPricing, calculateCost } from 'token-prices';

// Get pricing for a model
const result = await getModelPricing('openai', 'gpt-4o');
console.log(result.pricing.input);  // 2.5 ($/M tokens)

// Calculate cost
const cost = await calculateCost('anthropic', 'claude-sonnet-4', {
  inputTokens: 1500,
  outputTokens: 800,
});
console.log(cost.totalCost);  // 0.0165

Data Format

{
  "current": {
    "date": "2026-01-12",
    "models": {
      "gpt-4o": {
        "input": 2.5,
        "output": 10,
        "context": 128000
      }
    }
  },
  "previous": { ... }
}

Features

  • Daily automated price updates
  • Dual-date structure for handling update timing
  • Stale data detection
  • Cached input pricing support (Anthropic)
  • TypeScript types included
